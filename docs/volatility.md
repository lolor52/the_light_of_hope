# Волатильность — формулы

## Входные параметры

 
### Границы основной сессии
Определяйте начало и конец **основной торговой сессии** из MOEX ISS для борда инструмента (например, `TQBR`) через эндпоинт сессий борда. Аукционы открытия и закрытия исключайте из интервала расчёта. Не используйте жёстко заданные часы.


- **Тикер**: `T`
- **Дата основной сессии**: `D` в часовом поясе Europe/Moscow

## Данные (MOEX ISS)
- **candles 1m/5m**: `open`, `high`, `low`, `close`, `volume`
- **candles 1d**: `close` (для нормализации при необходимости)
- **Фильтр сессии**: используйте **только Основную торговую сессию** MOEX. Исключите утреннюю и вечернюю.
  - Часовой пояс: **Europe/Moscow (MSK)**
  - Временное окно основной сессии для большинства инструментов фондового рынка: **[10:00:00, 18:45:00)** MSK. Для укороченных/особых дней ориентируйтесь на биржевой календарь и фактические интервалы основной сессии.
  - Все дальнейшие расчёты выполняются по последовательности свечей `S_main(D)`, полученной после этой фильтрации для тикера `T` в дату `D`.

## Метрики
Все формулы ниже применяются **только к свечам `S_main(D)`**. Индекс `t` перебирает бары основной сессии по времени.
- Для `close_{t-1}` берите **предыдущий бар внутри `S_main`**. На первом баре торгового дня используйте **`close` последнего бара предыдущей основной сессии**.

```
TR_t = max( high_t - low_t,
            |high_t - close_{t-1}|,
            |low_t  - close_{t-1}| )

ATR_n   = SMA_n(TR_t)
ATR%_n  = ATR_n / close_t * 100

CumVol_t = Σ_{j=1..t} volume_j
RVOL_t   = CumVol_t / avg_{d=1..N}( CumVol_{d,t} )
```

Примечание по `RVOL_t`: сравнивайте **только по одинаковому индексу бара внутри основной сессии**. Если в дне `d` отсутствует бар с индексом `t` (укороченная сессия и т. п.), такой день **исключайте из среднего** `avg_{d=1..N}`.

## Нормировка и индекс волатильности по барам
Источником масштаба служит **историческое окно** по тому же тикеру и тому же индексу бара (time‑of‑day), а не текущая сессия.

Пусть \(H\) — последние \(K\) торговых дней до даты \(D\) (рекомендовано \(K=60\)). Для каждого индекса бара \(t\) рассчитайте:
- \(P5^{ATR\%}_t, P95^{ATR\%}_t\) — 5-й и 95-й перцентили распределения \(ATR\%_{d,t}\), \(d\in H\)
- \(P5^{RVOL}_t, P95^{RVOL}_t\) — 5-й и 95-й перцентили распределения \(RVOL_{d,t}\), \(d\in H\)

Нормировка:
\[
\widetilde{ATR\%}_t=\mathrm{clip}\Big(\frac{ATR\%_t-P5^{ATR\%}_t}{P95^{ATR\%}_t-P5^{ATR\%}_t+\varepsilon},0,1\Big),\quad
\widetilde{RVOL}_t=\mathrm{clip}\Big(\frac{RVOL_t-P5^{RVOL}_t}{P95^{RVOL}_t-P5^{RVOL}_t+\varepsilon},0,1\Big)
\]

Индекс волатильности бара:
\[
V_t = 100\,\big(0.6\,\widetilde{ATR\%}_t + 0.4\,\widetilde{RVOL}_t\big)
\]

Где \(\varepsilon\) — маленькая константа во избежание деления на ноль. Перцентили устойчивее min/max и сопоставимы между датами.

## Итоговая  оценка за сессию
Оценка волатильности тикера `T` за основную сессию даты `D` — усреднение по барам внутри `S_main(D)`:
```
V_session(D) = mean_{i ∈ S_main(D)}( V_i )
```
Диапазон результата: **0–100**.

## Параметры
```
ε = 1e-9
n = 14..20  (период ATR)
N = 20      (история для RVOL)
Таймфрейм: 1m или 5m
Сессия: только Основная (MSK), окно [T_open, T_close) по расписанию ISS; утренняя и вечерняя исключены
```


#### Первый бар сессии и междневной гэп
Для расчёта **внутрисессионной** волатильности исключайте междневной гэп:  
Для первого бара основной сессии используйте внутрисессионный вариант: TR_{t0} = max{H_{t0}-L_{t0}, |H_{t0}-O_{t0}|, |L_{t0}-O_{t0}|}. Овернайт‑гэп не учитывается в TR первого бара.
