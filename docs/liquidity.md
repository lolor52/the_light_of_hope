# Ликвидность — расчёт за Основную сессию MOEX

> Все расчёты выполняются **только по Основной торговой сессии MOEX**. Вечерняя, промежуточные и внебиржевые сессии исключены. **Аукционы открытия и закрытия также исключены.**

## Входные параметры
```
Ticker = <код тикера, один>
Date   = <YYYY-MM-DD> (MSK)
S      = множество тикеров для кросс-нормировки на Date
```
> Все расчёты выполняются **только по Основной торговой сессии MOEX**. Вечерняя, промежуточные и внебиржевые сессии исключены. **Аукционы открытия и закрытия также исключены.**

## Источник данных
- Только **MOEX ISS HTTP API** (`iss.moex.com`).
- Используемые конечные точки: `securities`, `candles` (interval=1), `boards/schedule`.
- Потоковые и сторонние источники не используются.

## Параметры
```
ε = 1e-9
MainSessionOnly = true
ExcludeAuctions = true
TZ = MSK (UTC+3)
```

## Выход
Одно число `L_i(Ticker, Date) ∈ [0, 100]` — оценка ликвидности.
При необходимости можно возвращать промежуточные сессионные метрики.
