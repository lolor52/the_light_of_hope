# Формулы «плоского» тренд‑фильтра (TQBR)

## Обозначения
- Минутные индексы: `t = 1..N`.
- Бар: `H_t, L_t, C_t, V_t` — максимум, минимум, закрытие, объём.
- Типичная цена:  
  **[1]** `P_typ(t) = (H_t + L_t + C_t) / 3`.
- Кумулятивный VWAP:  
  **[2]** `VWAP_t = ( Σ_{i=1..t} V_i * P_typ(i) ) / max( Σ_{i=1..t} V_i , ε )`.
- Сессионный диапазон:  
  **[3]** `R = max_t(H_t) − min_t(L_t)`, далее `R ← max(R, ε)`.
- Функция знака: `sgn(x) ∈ {−1,0,1}`. Индикатор: `1(·)`.
- Длительность выборки в минутах: `T_min` (для минутных баров `T_min = N`).
- Число машинного нуля: `ε = 1e−9`.

### Уточнения к \[2]–\[3]
- Если `V_i = 0`, вклад в числитель нулевой, знаменатель не увеличивается.
- Защита от деления на ноль реализуется через `max(·, ε)` для VWAP и `R`.

---

## Компонент 1. Пересечения с VWAP
**[4]** `K = Σ_{t=2..N} 1( sgn(C_t − VWAP_t) ≠ sgn(C_{t−1} − VWAP_{t−1}) )`

**[5]** `K_ref = floor( T_min / 45 )`, минимум `1`.

**[6]** `S_cross = 100 * min( 1 , K / max(1, K_ref) )`.

**Уточнение к \[4]**: если `sgn(C_t − VWAP_t) = 0`, для сравнения используется ближайший предыдущий ненулевой знак; при его отсутствии переход пропускается.

---

## Компонент 2. Доля времени у VWAP
Полоса шириной `b * R`, где `b > 0` — параметр.
  
**[7]** `F = (1/N) * Σ_{t=1..N} 1( |C_t − VWAP_t| ≤ b * R )`

**[8]** `S_band = 100 * F`.

---

## Компонент 3. Сессионный дрейф
**[9]** `D = | C_N − C_1 | / R`

**[10]** `S_drift = 100 * ( 1 − min(1, D) )`.

---

## Итоговый скор
**[11]** `S = clip_{[0,100]}( 0.4*S_cross + 0.4*S_band + 0.2*S_drift )`.

### Уточнения к \[11]
- `clip_{[0,100]}(x) = min(100, max(0, x))`.
- Итог `S ∈ [0, 100]`.
