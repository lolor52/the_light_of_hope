# Справочник формул

Этот файл содержит основные формулы, используемые при расчетах внутри проекта `invest_intraday`. Обозначения и формулы подобраны так, чтобы можно было напрямую использовать их в коде и при проверке результатов.

## Базовые обозначения
- $p_i$ — цена сделки на шаге $i$.
- $v_i$ — объем сделки на шаге $i$.
- $n$ — количество сделок (баров, тиков) в рассматриваемом интервале.
- $P_t$ — цена инструмента в момент времени $t$.
- $V_t$ — накопленный объем к моменту времени $t$.
- $\Delta t$ — размер временного шага между наблюдениями.
- $VWAP_t$ — значение VWAP на момент времени $t$.

## Объемно-взвешенная средняя цена (VWAP)
\[
VWAP_t = \frac{\sum_{i=1}^{n} p_i \cdot v_i}{\sum_{i=1}^{n} v_i}
\]

Если требуется инкрементально обновлять VWAP при поступлении новой сделки с ценой $p_{n+1}$ и объемом $v_{n+1}$, можно использовать рекуррентную форму:
\[
VWAP_{t+\Delta t} = \frac{VWAP_t \cdot V_t + p_{n+1} \cdot v_{n+1}}{V_t + v_{n+1}}
\]

Где $V_t = \sum_{i=1}^{n} v_i$.

## Value Area (VAL, VAP, VAH)
Для расчета значений Value Area необходимо построить профиль объема по ценовым уровням.

1. Разбейте ценовой диапазон на дискретные уровни $L_j$ с шагом $\Delta p$.
2. Для каждого уровня накопите объем:
\[
Vol(L_j) = \sum_{i \in L_j} v_i
\]
3. Найдите ценовой уровень с максимальным объемом — это уровень Point of Control (POC).
4. Добавляйте симметричные уровни выше и ниже POC, пока суммарный объем не достигнет выбранной доли (обычно 70%):
\[
\sum_{L_j \in \text{VA}} Vol(L_j) \geq 0{,}7 \cdot \sum_j Vol(L_j)
\]
5. Значения Value Area Low (VAL) и Value Area High (VAH) берутся как минимальный и максимальный уровни внутри полученного диапазона.

## Проверка условия VAH > VWAP > VAL
После получения VWAP, VAL и VAH проверка выполняется прямым сравнением:
\[
VAH > VWAP > VAL
\]

## Дневной VWAP
Для расчета дневного VWAP используйте все сделки с начала торговой сессии до текущего момента:
\[
VWAP^{day}_t = \frac{\sum_{\tau=\text{session start}}^{t} p_{\tau} \cdot v_{\tau}}{\sum_{\tau=\text{session start}}^{t} v_{\tau}}
\]

## Скользящая средняя цены
Для удобства сравнения VWAP с простыми скользящими средними можно использовать формулу SMA:
\[
SMA_t = \frac{1}{m} \sum_{k=0}^{m-1} P_{t-k}
\]
где $m$ — длина окна.

## Экспоненциальная скользящая средняя объема
\[
EMA^{(vol)}_t = \alpha \cdot v_t + (1-\alpha) \cdot EMA^{(vol)}_{t-1}
\]
где $\alpha = \frac{2}{m+1}$ и $m$ — размер окна в барах.

## Расчет стандартного отклонения цены
Для оценки волатильности внутри интервала используйте стандартное отклонение:
\[
\sigma_t = \sqrt{\frac{1}{n} \sum_{i=1}^{n} (p_i - \bar{p})^2}
\]
где $\bar{p} = \frac{1}{n} \sum_{i=1}^{n} p_i$.

Эти формулы можно комбинировать для построения более сложных индикаторов и сигналов, используемых в проекте.
