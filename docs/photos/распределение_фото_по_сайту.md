# Предварительное распределение фото по сайту

## Важное ограничение

Этот документ составлен по именам файлов, ориентации, размерам и типу материалов. Полноценного визуального просмотра локальных фото в текущей среде нет. Поэтому рекомендации ниже являются рабочими и безопасными, но финально их стоит утвердить после ручной визуальной проверки самих кадров.

Также в папке media обнаружено 11 файлов с фотоматериалами и отдельная папка media/logos с логотипами. Если под «12 фото» подразумевается ещё один файл из media/logos, его нужно отдельно подтвердить.

## Единый стиль обработки

### Что делать

- слегка выровнять экспозицию между кадрами;
- немного смягчить резкие цветовые различия между файлами;
- при необходимости чуть снизить перенасыщенность;
- сохранить естественные оттенки кожи, пространства и предметов;
- использовать одинаковые радиусы скругления и одинаковые пропорции контейнеров в интерфейсе.

### Чего не делать

- не усиливать контраст до «глянца»;
- не использовать тяжёлые пресеты;
- не делать драматичное затемнение;
- не применять художественные фильтры;
- не вырезать людей из фона и не собирать коллажи;
- не искажать пропорции и не растягивать вертикальные фото под горизонтальные блоки.

Для окончательного выбора без риска ошибки нужно визуально проверить сами кадры. Оптимальный вариант: приложить контакт-лист из 11 фото в чат или прислать 11 миниатюр одним изображением.
