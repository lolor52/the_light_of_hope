# ZigZag с адаптивным порогом по ATR и фильтром прохождения через VWAP — формулы и уточнения

## Область определения и индексация
- Пусть после фильтра основной сессии имеем минутные бары \(t=1,\dots,T\) с полями \(O_t,H_t,L_t,C_t,V_t\).
- В первом баре примите \(C_0 = C_1\). Индексы пивотов обозначаются \(\tau_j\).

## Базовые величины
Типичная цена:
\[
TP_t=\frac{H_t+L_t+C_t}{3}.
\]

Инкрементальный VWAP:
\[
VWAP_t=\frac{\sum_{i=1}^{t} TP_i\cdot V_i}{\sum_{i=1}^{t} V_i},\quad
\text{если } \sum_{i=1}^{t} V_i=0:\ VWAP_t=VWAP_{t-1},\ VWAP_1=TP_1.
\]

True Range:
\[
TR_t=\max\!\big(H_t-L_t,\ |H_t-C_{t-1}|,\ |L_t-C_{t-1}|\big),\quad TR_1=H_1-L_1.
\]

ATR (Вайлдер):
\[
ATR_N=\frac{1}{N}\sum_{i=1}^{N}TR_i,\qquad
ATR_t=\frac{(N-1)\cdot ATR_{t-1}+TR_t}{N}\quad(t>N).
\]
Уточнение: если \(T<N\), используйте \(\frac{1}{T}\sum_{i=1}^{T}TR_i\).

Адаптивный порог:
\[
\theta_t=K\cdot ATR_t,\quad K>0.
\]

## ZigZag с порогом \(\theta_t\)
### Инициализация направления и первого пивота
Найдите
\[
t_0=\min\{t\ge2:\ |C_t-C_1|\ge \theta_t\}.
\]
Если \(C_{t_0}>C_1\) — стартовая нога **UP**, стартовый пивот
\[
P_1=\big(\min L_{1..t_0},\ \tau_1=\arg\min L_{1..t_0}\big)\ \text{(LOW)}.
\]
Иначе — **DOWN**, стартовый пивот
\[
P_1=\big(\max H_{1..t_0},\ \tau_1=\arg\max H_{1..t_0}\big)\ \text{(HIGH)}.
\]

### Правила обновления экстрем и разворота
Нотация текущей незавершённой экстремы: \(H_t^*\) (для UP) или \(L_t^*\) (для DOWN) и её время \(\tau^*\).

- Нога **UP**:
  \[
  H_t^*=\max(H_{t-1}^*,H_t);\quad
  \text{разворот при } H_t^*-C_t\ge \theta_t.
  \]
  Тогда фиксируется пивот \(P_j=(H_{\tau^*}^*,\tau^*)\) типа HIGH, направление меняется на **DOWN**, сброс \(L_t^*=L_t,\ \tau^*=t\).

- Нога **DOWN**:
  \[
  L_t^*=\min(L_{t-1}^*,L_t);\quad
  \text{разворот при } C_t-L_t^*\ge \theta_t.
  \]
  Тогда фиксируется пивот \(P_j=(L_{\tau^*}^*,\tau^*)\) типа LOW, направление меняется на **UP**, сброс \(H_t^*=H_t,\ \tau^*=t\).

Уточнения:
- В условиях разворота используется \(C_t\) против экстремы \(H^*_t/L^*_t\).
- При равенстве экстрем обновляйте \(\tau^*\) на более поздний бар.
- Два пивота подряд одного типа не допускаются.

## Фильтр валидности свинга: пересечение VWAP
Рассматривайте каждый отрезок между соседними пивотами \([P_{j-1}\!\to\!P_j]\) на интервале баров \((\tau_{j-1},\dots,\tau_j]\). Свинг валиден, если
\[
\exists\, t\in(\tau_{j-1}\!+\!1,\dots,\tau_j]:\ (C_{t-1}-VWAP_{t-1})\cdot(C_t-VWAP_t)\le 0
\]
и не одновременно \(C_{t-1}=VWAP_{t-1}\) и \(C_t=VWAP_t\).

Опциональная уточняющая проверка линейной интерполяцией:
\[
\tilde{C}(u)=C_{t-1}+u\,(C_t-C_{t-1}),\quad
\widetilde{VWAP}(u)=VWAP_{t-1}+u\,(VWAP_t-VWAP_{t-1}),\ u\in[0,1].
\]
Пересечение существует, если \(\tilde{C}(u)=\widetilde{VWAP}(u)\) для некоторого \(u\in(0,1]\), что эквивалентно знакопеременности разности на концах.

## Итоговая величина
Пусть \(S\) — число валидных свингов среди отрезков \([P_1\!\to\!P_2],\dots,[P_{m-1}\!\to\!P_m]\).
\[
\boxed{\text{SwingCountPairs}=\left\lfloor \frac{S}{2}\right\rfloor}.
\]
Уточнение: если \(S<2\), возвращается \(0\).
