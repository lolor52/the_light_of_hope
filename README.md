# Свет Надежды

Одностраничный сайт-визитка для НКО «Свет Надежды» на чистом HTML, CSS и JavaScript без Node.js и внешних библиотек.

## Ограничение по оплате

В рамках текущего проекта рассматриваются только сценарии приема пожертвований без backend и без базы данных.

Допустимые варианты для сайта:

- только статический QR-код СБП на стороне сайта;
- кнопка или ссылка на заранее созданную внешнюю страницу оплаты ЮKassa.

Недопустимые варианты в рамках этого репозитория:

- создание платежей через API ЮKassa из браузера;
- хранение секретного ключа ЮKassa на клиенте;
- проверка факта успешной оплаты только по данным frontend.

## Структура проекта

- [index.html](index.html) — главная HTML-страница сайта
- [css](css) — стили сайта
- [js](js) — JavaScript для мобильного меню
- [media](media) — локальные изображения и логотип
- [docs/payment/сценарий_приема_пожертвований.md](docs/payment/сценарий_приема_пожертвований.md) — правила и ограничения по оплате без backend

## Как открыть сайт

1. Открыть файл [index.html](index.html) в браузере.
2. Для локальной проверки можно использовать любой простой статический сервер, но это не обязательно.

## Где редактировать сайт

- Тексты и структура блоков: [index.html](index.html)
- Внешний вид, адаптивность и кнопки: [css/styles.css](css/styles.css)
- Поведение мобильного меню: [js/main.js](js/main.js)
- Фото и логотип: [media](media) и [media/logos](media/logos)
